# Heexy Bot (Crawler)

**Heexybot** je náš oficiální webový crawler, který navštěvuje webové stránky a indexuje jejich obsah pro **vyhledávač Heexy**.

Díky Heexybotovi můžeme poskytovat rychlé, přesné a aktuální výsledky vyhledávání pro naše uživatele.

---

### 🔍 Co Heexybot dělá?

- 📥 Shromažďuje a indexuje **veřejně dostupný webový obsah**  
- 🚫 **Blokuje škodlivé weby** a nekvalitní zdroje  
- 🧠 Zaměřuje se pouze na **relevantní a důležitý** obsah, čímž snižuje množství spamu  
- 📡 Pomáhá budovat a aktualizovat index vyhledávače Heexy  
- 🔐 Respektuje pravidla webů (`robots.txt`)  
- 🤝 Dodržuje etické zásady crawlování  

---

### 📌 Důležité informace

- Heexybot **nesleduje** ani **nesbírá osobní údaje**  
- Funguje převážně z **evropských serverů**  
- Identifikuje se vlastním **User-Agentem**  
- Upřednostňuje **kvalitu**, **bezpečnost** a **uživatelský komfort**
