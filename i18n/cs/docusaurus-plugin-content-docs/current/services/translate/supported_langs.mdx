# Podporované jazyky

<img src="/img/docs/langauges.png" style={{ borderRadius: '0.5rem' }} />

Vítejte v článku o **Heexy Translate**!  
Heexy Translate je výkonná překladová služba, která nabízí rychlé a přesné překlady mezi jazyky. V tomto článku se dozvíte, jaké jazyky Heexy Translate podporuje, a které zatím ne.

Heexy Translate je moderní nástroj navržený pro **bezproblémovou komunikaci mezi různými jazyky**. Díky pokročilým algoritmům a nejnovějším technologiím poskytuje rychlé a přesné překlady, což z něj dělá nepostradatelný nástroj pro jednotlivce i firmy.

## Podporované a nepodporované jazyky

V tomto článku najdete **rozsáhlý seznam jazyků**, které Heexy Translate podporuje.  
Ať už chcete překládat z angličtiny do španělštiny nebo z čínštiny do francouzštiny, Heexy Translate vám pomůže.  
Zároveň uvedeme i jazyky, které momentálně podporovány nejsou – pro kompletní přehled možností služby.

Zůstaňte s námi a objevujte svět překladů s Heexy Translate!
