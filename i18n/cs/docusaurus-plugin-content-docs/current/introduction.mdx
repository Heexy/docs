# Vítejte v Heexy!

Vítejte v dokumentaci Heexy

Heexy je vyhledávač, který klade důraz na soukromí a pohodlí uživatelů. Jde o platformu bez reklam, navrženou pro bezpečné vyhledávání. Naším hlavním cílem je zajistit maximální soukromí a komfort při používání Heexy. Aktivně nasloucháme komunitě a zapojujeme ji do vylepšování našeho vyhledávače.

## Funkce

Heexy nabízí širokou škálu služeb pro lepší zážitek z vyhledávání. Mezi klíčové funkce patří:

🔎 **Bezpečné vyhledávání**: Heexy chrání soukromí uživatelů a zajišťuje důvěrnost vašich dotazů.

🚫 **Bez reklam**: Rozlučte se s rušivými reklamami! Heexy poskytuje prostředí bez reklam.

✅ **Uživatelsky přívětivé rozhraní**: Intuitivní rozhraní pro pohodlné a příjemné vyhledávání.

❤️ **Podpora komunity**: Nasloucháme komunitě a sbíráme zpětnou vazbu pro neustálé zlepšování našich služeb.

## Začínáme

Ať už jste nováček nebo stávající uživatel, tato dokumentace vám poskytne kompletní informace o našich službách, nastavení účtu, integraci API a dalším.  
Prozkoumejte dokumentaci a využijte Heexy naplno. Máte-li dotazy nebo zpětnou vazbu, neváhejte kontaktovat náš tým podpory.
