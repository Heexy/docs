# Podporované jazyky

Zde jsou informace o jazycích, které jsou podporovány a přeloženy

Překlad našich služeb je pro nás velmi důležitý, snažíme se překládat vše do co nejvíce jazyků a snažíme se najít a spolupracovat s rodilými mluvčími a naší komunitou, aby byly překlady co nejpřesnější. Chceme, aby všichni na světě měli přístup k Heexy ve svém rodném jazyce, protože Heexy je pro celý svět.
